4   Asset Pack Editor¶

In a Phaser game you load the files using any of the `Phaser.Loader`_ methods. Many of these methods require additional information to describe the format of the file:

``` this.load.spritesheet({

key: “player.png”, width: 16, height: 16

});

```

There is a especial method, the `load.pack()`_ that allows to load the description of the files from a JSON file: the Asset Pack File.
